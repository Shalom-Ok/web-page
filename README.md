# Weather App 🌤️

A simple and responsive weather application built using **HTML, CSS, and JavaScript**.

This project allows users to check weather conditions in a clean and user-friendly interface. It was built as part of my journey into software engineering and front-end development.

## Features
- Search weather by location
- Clean and responsive UI
- Styled with custom CSS
- Front-end project
- Focus on layout, styling, and user experience

## Technologies Used
- HTML
- CSS
- JavaScript

## Future Improvements
- Integrate live weather API data
- Add forecast for multiple days
- Improve mobile responsiveness
- Add loading states and error handling
